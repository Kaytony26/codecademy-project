Display: block

This type of display is for block-level elements. such as <h1-h6>, <p>, <div> and <footer>.

Here are some facts about them:

  • The elements with this display are not displayed on the same line as the content around them.
  • They start on a new line
  • They fill the entire width of the page by default, but you can set their width property.
  • The syntax for this is: p {display: block;}

Display: inline

This type of display is for inline elements such as, <em>, <strong> and <a>.

Here are some facts about them:

  • These elements have a box that wraps tightly around their content, only taking up the amount of space necessary to display their content.
  • They do not requirea new line
  • They can be displayed on the same line as their surrounding text
  • The syntax for this is: p {display: inline;}

Display: inline-block

This type of display is for inline-block elements like, <img>.

Here are some facts about them:

  • These elements can appear next to each other.
  • You can specify their dimensions using width and height properties
  • You can be specific as to how much space you want them to take up.
  • The syntax for this is: p {display: inline-block;}
